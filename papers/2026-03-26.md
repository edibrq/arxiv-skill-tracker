### 1. AVO: Agentic Variation Operators for Autonomous Evolutionary Search

**Source:** arXiv  
**ID:** arXiv:2603.24517  
**URL:** https://arxiv.org/abs/2603.24517  
**Authors:** Xu Bing, Terry Chen, Zhifan Ye (co-first authors), NVIDIA Research  
**Date:** 2026-03-26  
**Type:** Preprint  

**Abstract/Summary:**  
NVIDIA introduces Agentic Variation Operators (AVO), a new type of evolutionary variation operator that replaces fixed mutation, crossover, and manually designed heuristics in classical evolutionary search with autonomous coding agents. Unlike traditional LLM-based evolutionary search where the model is restricted to candidate generation within a preset pipeline, AVO promotes the agent to a self-directed mutation operator that can autonomously consult domain knowledge, implement modifications, and verify results through iterative environment interaction.

The researchers applied AVO to optimize the multi-head attention (MHA) kernel on NVIDIA Blackwell B200 GPU. In a continuous 7-day autonomous evolution without human intervention, the agent explored 500+ optimization directions and evolved 40 kernel versions. The final MHA kernel achieved 1668 TFLOPS at BF16 precision, surpassing NVIDIA's expert-optimized cuDNN by up to 3.5% and FlashAttention-4 by up to 10.5%. The optimization techniques transferred effectively to grouped query attention (GQA) with only 30 minutes of additional autonomous adaptation.

Key optimizations discovered by the agent include branchless accumulator rescaling, tensor core pipeline overlap, and register rebalancing across warp groups—demonstrating real hardware-level reasoning rather than superficial code transformation.
