### 1. ARISE: Agent Reasoning with Intrinsic Skill Evolution in Hierarchical Reinforcement Learning

- **Source:** arXiv
- **ID:** arXiv:2603.16060
- **Authors:** Yu Li, Rui Miao, Zhengling Qi, Tian Lan
- **Date:** 2026-03-17 (v1), 2026-03-18 (v2)
- **Type:** Preprint
- **URL:** https://arxiv.org/abs/2603.16060
- **DOI:** 10.48550/arXiv.2603.16060

**Abstract/Summary:**
Introduces ARISE, a hierarchical RL framework where a shared policy operates as both Skills Manager (high-level) and Worker (low-level). The Manager maintains a tiered skill library through structured summarization of successful solution traces, employing policy-driven selection to retrieve relevant skills for future rollouts. A hierarchical reward design guides co-evolution of reasoning ability and library quality. Experiments on seven benchmarks show ARISE consistently outperforms GRPO-family algorithms and memory-augmented baselines, with notable gains on out-of-distribution tasks.

### 2. Memento-Skills: Let Agents Design Agents

- **Source:** arXiv
- **ID:** arXiv:2603.18743
- **Authors:** Huichi Zhou, Siyuan Guo, Anjie Liu, Zhongwei Yu, Ziqin Gong, Bowen Zhao, Zhixun Chen, Menglong Zhang, Yihang Chen, Jinsong Li, Runyu Yang, Qiangbin Liu, Xinlei Yu, Jianmin Zhou, Na Wang, Chunyang Sun, Jun Wang
- **Date:** 2026-03-19
- **Type:** Preprint
- **URL:** https://arxiv.org/abs/2603.18743

**Abstract/Summary:**
Introduces Memento-Skills, a generalist continually-learnable LLM agent system that functions as an agent-designing agent: it autonomously constructs, adapts, and improves task-specific agents through experience. Built on memory-based RL with stateful prompts, where reusable skills (stored as structured markdown files) serve as persistent, evolving memory. Uses Read-Write Reflective Learning mechanism: in read phase, skill router selects relevant skill; in write phase, agent updates skill library based on experience. Enables continual learning without updating LLM parameters. Achieves 26.2% and 116.2% relative improvements on General AI Assistants benchmark and Humanity's Last Exam respectively.

### 3. MetaClaw: Just Talk – An Agent That Meta-Learns and Evolves in the Wild

- **Source:** arXiv
- **ID:** arXiv:2603.17187
- **Authors:** Pangshai, Janin
- **Date:** 2026-03-17
- **Type:** Preprint
- **URL:** https://arxiv.org/abs/2603.17187

**Abstract/Summary:**
Presents MetaClaw, a continual meta-learning framework that jointly evolves a base LLM policy and library of reusable behavioral skills. Employs two mechanisms: (1) Skill-driven fast adaptation analyzes failure trajectories via LLM evolver to synthesize new skills for immediate improvement with zero downtime; (2) Opportunistic policy optimization performs gradient-based updates via cloud LoRA fine-tuning and RL-PRM, triggered during user-inactive windows by Opportunistic Meta-Learning Scheduler (OMLS). Built on proxy-based architecture, scales to production-size LLMs without local GPUs. Experiments show skill-driven adaptation improves accuracy by up to 32% relative; full pipeline advances Kimi-K2.5 accuracy from 21.4% to 40.6%.

### 4. SLEA-RL: Step-Level Experience Augmented Reinforcement Learning for Multi-Turn Agentic Training

- **Source:** arXiv
- **ID:** arXiv:2603.18079
- **Authors:** Prince Zizhuang Wang, Shuli Jiang
- **Date:** 2026-03-18
- **Type:** Preprint
- **URL:** https://arxiv.org/abs/2603.18079
- **DOI:** 10.48550/arXiv.2603.18079

**Abstract/Summary:**
Proposes SLEA-RL, a framework that retrieves relevant experiences at each decision step conditioned on current observation (vs. static retrieval once at episode start). Three components: (i) step-level observation clustering for efficient cluster-indexed retrieval; (ii) self-evolving experience library that distills successful strategies and failure patterns through score-based admission; (iii) policy optimization with step-level credit assignment for fine-grained advantage estimation. Experience library evolves alongside policy through semantic analysis rather than gradient updates. Demonstrates superior performance on long-horizon multi-turn agent benchmarks.
