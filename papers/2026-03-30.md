### 1. GUIDE: Resolving Domain Bias in GUI Agents through Real-Time Web Video Retrieval and Plug-and-Play Annotation

- **Source:** arXiv
- **ID:** arXiv:2603.26266
- **URL:** https://arxiv.org/abs/2603.26266
- **Authors:** Rui Xie, Zhi Gao, Chenrui Shi, Zirui Shang, Lu Chen, Qing Li
- **Date:** 2026-03-30
- **Type:** Preprint
- **Subjects:** cs.AI, cs.CV

**Abstract:**  
Large vision-language models have endowed GUI agents with strong general capabilities for interface understanding and interaction. However, due to insufficient exposure to domain-specific software operation data during training, these agents exhibit significant domain bias. This paper presents GUIDE (GUI Unbiasing via Instructional-Video Driven Expertise), a training-free, plug-and-play framework that resolves GUI agent domain bias by autonomously acquiring domain-specific expertise from web tutorial videos through a retrieval-augmented automated annotation pipeline. GUIDE introduces a subtitle-driven Video-RAG pipeline and a fully automated annotation pipeline built on an inverse dynamics paradigm. Extensive experiments on OSWorld demonstrate GUIDE's generality, consistently yielding over 5% improvements without modifying any model parameters.

**Key Contributions:**
- Autonomous skill acquisition from web tutorial videos
- Training-free, plug-and-play framework for GUI agents
- Video-RAG pipeline for domain-specific expertise retrieval
- Automated annotation via inverse dynamics paradigm

### 2. AIRA_2: Overcoming Bottlenecks in AI Research Agents

- **Source:** arXiv
- **ID:** arXiv:2603.26499
- **URL:** https://arxiv.org/abs/2603.26499
- **Authors:** Karen Hambardzumyan, Nicolas Baldwin, Edan Toledo, Rishi Hazra, Michael Kuchnik, Bassel Al Omari, Thomas Simon Foster, Anton Protopopov, Jean-Christophe Gagnon-Audet, Ishita Mediratta, Kelvin Niu, Michael Shvartsman, Alisia Lupidi, Alexis Audran-Reiss, Parth Pathak, Tatiana Shavrina, Despoina Magka, Hela Momand, Derek Dunfield, Nicola Cancedda, Pontus Stenetorp, Carole-Jean Wu, Jakob Nicolaus Foerster, Yoram Bachrach, Martin Josifoski
- **Date:** 2026-03-30
- **Type:** Preprint
- **Subjects:** cs.AI

**Abstract:**  
Existing research has identified three structural performance bottlenecks in AI research agents: (1) synchronous single-GPU execution constrains sample throughput, (2) a generalization gap where validation-based selection causes performance to degrade over extended search horizons, and (3) the limited capability of fixed, single-turn LLM operators imposes a ceiling on search performance. AIRA_2 addresses these through: an asynchronous multi-GPU worker pool that increases experiment throughput linearly, a Hidden Consistent Evaluation protocol for reliable evaluation signals, and ReAct agents that dynamically scope their actions and debug interactively. On MLE-bench-30, AIRA_2 achieves a mean Percentile Rank of 71.8% at 24 hours and steadily improves to 76.0% at 72 hours.

**Key Contributions:**
- Asynchronous multi-GPU architecture for autonomous agents
- ReAct agents with dynamic action scoping (skill execution)
- Hidden Consistent Evaluation protocol
- Improved agent performance through architectural innovations

### 3. CADSmith: Multi-Agent CAD Generation with Programmatic Geometric Validation

- **Source:** arXiv
- **ID:** arXiv:2603.26512
- **URL:** https://arxiv.org/abs/2603.26512
- **Authors:** Jesse Barkley, Rumi Loghmani, Amir Barati Farimani
- **Date:** 2026-03-30
- **Type:** Preprint
- **Subjects:** cs.AI
- **Comments:** 8 pages, 6 figures

**Abstract:**  
Existing methods for text-to-CAD generation either operate in a single pass with no geometric verification or rely on lossy visual feedback that cannot resolve dimensional errors. CADSmith is a multi-agent pipeline that generates CadQuery code from natural language and undergoes iterative refinement through two nested correction loops: an inner loop resolving execution errors and an outer loop grounded in programmatic geometric validation. The system uses retrieval-augmented generation over API documentation rather than fine-tuning, maintaining a current database as the underlying CAD library evolves. CADSmith achieves 100% execution rate (up from 95%), improves median F1 score from 0.9707 to 0.9846, and reduces mean Chamfer Distance from 28.37 to 0.74.

**Key Contributions:**
- Multi-agent pipeline with iterative skill refinement
- Programmatic geometric validation for agent feedback
- Retrieval-augmented generation over evolving API documentation
- Closed-loop refinement with geometric feedback
