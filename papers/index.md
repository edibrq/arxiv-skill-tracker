## Usage

Before adding new content:
1. Search this index for the unique ID (arXiv ID, DOI, or URL slug)
2. If found → skip (already tracked)
3. If not found → add to daily file AND update this index

**Type values:** `Paper` (peer-reviewed), `Preprint` (arXiv), `Article` (blog/Medium), `Code` (Papers with Code)

**References:** List arXiv IDs/DOIs of papers cited by this work. Use `-` if none tracked. When adding a new paper, extract its references and check if any are already in the index (for citation network tracking).
