## 1. SkillRL: Evolving Agents via Recursive Skill-Augmented Reinforcement Learning

**arXiv:** [2602.08234](https://arxiv.org/abs/2602.08234)  
**Published:** 2026-02-09  
**Authors:** Peng Xia, Jianwen Chen, Hany Wang, Jiaqi Liu, Kaide Zeng, Yu Wang, Siwei Han, Yiyang Zhou, Xujiang Zhao, Haifeng Chen, Zeyu Zheng, Cihang Xie, Huaxiu Yao  
**Code:** [github.com/aiming-lab/SkillRL](https://github.com/aiming-lab/SkillRL)

### Abstract
Large Language Model (LLM) agents have shown stunning results in complex tasks, yet they often operate in isolation, failing to learn from past experiences. Existing memory-based methods primarily store raw trajectories, which are often redundant and noise-heavy. This prevents agents from extracting high-level, reusable behavioral patterns that are essential for generalization.

In this paper, we propose **SkillRL**, a framework that bridges the gap between raw experience and policy improvement through automatic skill discovery and recursive evolution. Our approach introduces:
- Experience-based distillation mechanism to build a hierarchical skill library (SkillBank)
- Adaptive retrieval strategy for general and task-specific heuristics
- Recursive evolution mechanism allowing the skill library to co-evolve with the agent's policy during reinforcement learning

**Results:** SkillRL achieves state-of-the-art performance on ALFWorld (89.9%), WebShop (72.7%), and seven search-augmented tasks, outperforming strong baselines by 15.3%.

## 2. AutoSkill: Experience-Driven Lifelong Learning via Skill Self-Evolution

**arXiv:** [2603.01145](https://arxiv.org/abs/2603.01145)  
**Published:** 2026-03-04  

### Abstract
In practical LLM applications, users repeatedly express stable preferences and requirements—such as reducing hallucinations, following institutional writing conventions, or avoiding overly technical wording—yet such interaction experience is seldom consolidated into reusable knowledge.

We present **AutoSkill**, an experience-driven lifelong learning framework that enables LLM agents to automatically derive, maintain, and reuse skills from dialogue and interaction traces. Key features:
- Skill extraction from interaction traces
- Iterative refinement and versioned evolution
- Dynamic injection into future requests
- Model-agnostic plug-in layer

AutoSkill turns ephemeral interaction experience into explicit, reusable, and composable capabilities.

## 3. Evolving Programmatic Skill Networks (PSN)

**arXiv:** [2601.03509](https://arxiv.org/abs/2601.03509)  
**Published:** 2026-01-07  
**Authors:** Haochen Shi, Xingdi Yuan, Bang Liu  

### Abstract
We study continual skill acquisition in open-ended embodied environments where an agent must construct, refine, and reuse an expanding library of executable skills.

We introduce the **Programmatic Skill Network (PSN)**, a framework in which skills are executable symbolic programs forming a compositional network that evolves through experience. PSN defines three core mechanisms:
1. **REFLECT** for structured fault localization over skill compositions
2. Progressive optimization with maturity-aware update gating
3. Canonical structural refactoring under rollback validation

The framework exhibits structural parallels to neural network training. Experiments on MineDojo and Crafter demonstrate robust skill reuse, rapid adaptation, and strong generalization.

## 4. EvoSkill: Automated Skill Discovery for Multi-Agent Systems

**arXiv:** [2603.02766](https://arxiv.org/abs/2603.02766)  
**Published:** 2026-03-03  
**Authors:** Salaheddin Alzubi, Noah Provenzano, Jaydon Bingham, Weiyuan Chen, Tu Vu  

### Abstract
Coding agents are increasingly used as general-purpose problem solvers, but their flexibility does not confer the domain expertise needed for specialized tasks.

We introduce **EvoSkill**, a self-evolving framework that automatically discovers and refines agent skills through iterative failure analysis:
- Analyzes execution failures
- Proposes new skills or edits to existing ones
- Materializes them into structured, reusable skill folders
- Pareto frontier governance for skill selection

**Results:**
- OfficeQA: 7.3% improvement (60.6% → 67.9%)
- SealQA: 12.1% gain (26.6% → 38.7%)
- Zero-shot transfer from SealQA to BrowseComp: +5.3%
